# Pr-ctica-Tema-4---Instal-laci-i-Configuraci-de-Moodle
En esta practica nos hemos instalado el moodle desde una maquina de IsardVDI y haremos el rol de administrados para configurar nuestro moddle

# 1. Configuració inicial del Moddle
En este lo que hice fue iniciar sesión como administrador y luego cambiamos la contraseña y correo como decir en los pasos de la instalación para seguir

![](https://github.com/GABRIELL410/Pr-ctica-Tema-4---Instal-laci-i-Configuraci-de-Moodle/blob/main/Captura%20de%20pantalla%20de%202026-04-10%2014-03-45.png)
en este apartado configuramos nuestro perfil. Ponemos nuestro correo y contraseña y el nombre de usuario que queremos que salga.

# 1.1 Cambiar correo y contraseña

![](https://github.com/GABRIELL410/Pr-ctica-Tema-4---Instal-laci-i-Configuraci-de-Moodle/blob/main/Captura%20de%20pantalla%20de%202026-04-10%2014-13-08.png)

Y cuando entremos bajamos y buscamos donde pongan las opciones para poder cambiar la longitud de la contraseña y los caracteres.
